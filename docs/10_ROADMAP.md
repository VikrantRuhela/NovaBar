# Nova Bar – Roadmap

This document outlines the current feature roadmap.

---

## 1. Completed

* **Dual Engine Core**: Overlay rendering via standard Application Window overlay and Accessibility overlay modes.
* **Notification Priority Handler**: Queue management system filtering call, GPS, media, timer, and battery updates.
* **Google Maps Maneuver Parser**: Reflection-based extraction of maneuver drawable resources.
* **OEM Clock Compatibility**: Parsers supporting Google, Samsung, MIUI/HyperOS, OnePlus, Oppo/Realme, Vivo, and Nothing clocks.
* **Call Handling**: Call status tracking and End Call integration (Telecom manager and Accessibility window-click fallbacks).
* **Luminance Adaptor**: Screenshot-based top-center color analyzer (for Android 11+ devices).
* **Split Camera Cutout Mode**: Custom layouts that split around centered punch-hole cameras.

---

## 2. In Progress

* **Visualizer Styling Options**: Expanding visualizer types (Waveform, Pulse, Minimal) and sensitivity tuning in Settings.
* **Allowed Package Filters**: Let users filter which apps can post notifications to the overlay.

---

## 3. Planned

* **Synchronization Accuracy**: Improve timer precision to minimize drifts.
* **Bypass Guide Dashboard**: Add in-app guides helping users whitelist Nova Bar from aggressive battery optimization software.
* **New OEM Clock Profiles**: Add support for minor Android clock apps (e.g. Motorola, Sony, Asus).
