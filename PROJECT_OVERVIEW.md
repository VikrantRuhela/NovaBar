# Project Overview

**NovaBar** is an Android utility overlay that provides a floating pill‑style control bar (the *Nova Bar*) on top of other apps. It offers quick toggles for:
- Enabling/disabling the overlay engine
- Syncing with the system status bar visibility
- Accessing notification & media controls
- Managing accessibility‑based color matching
- Phone call handling directly from the pill

The app is built with **Kotlin**, **Jetpack Compose**, and a modular architecture separating data, domain, presentation, services, UI components, and utilities.

Key goals:
- Minimal intrusiveness – works over any app without requiring root.
- Highly customizable appearance (size, opacity, gravity, offsets).
- Responsive UI with smooth animations and micro‑interactions.
- Secure permission handling (overlay, notification listener, accessibility, phone call).
